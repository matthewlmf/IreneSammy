== Description ==

Custom Social Icon is used to set social links with images respectively!
Generally, In header and footer, we need to assign social links associated with your selected images. 
By using this plugin, you can set your own links with your selected image from media.



== Installation ==

1. Upload the 'custom_social_icon' folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to settings -> Custom Social Icons
4. add your links with your selected images
5. Use the shortcode `[CSI]` in your posts or pages or template.
